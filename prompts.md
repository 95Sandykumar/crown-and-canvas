## Tips for Best Results
- **Upload the `before.webp`** as reference/input image so the AI preserves the pet's face, fur color, and features
- **Model**: Nano Banana Pro on Higgsfield
- **Aspect ratio**: Select **3:4** in the UI (do NOT include ratio in the prompt)
- **Resolution**: 1K
- **Images**: Generate 4 at a time, pick the best one
- **Save as**: `after.webp` in the matching folder (e.g., `public/portraits/renaissance-king/after.webp`)
- If the tool doesn't output WebP, save as PNG/JPG and convert to WebP using any free online converter
